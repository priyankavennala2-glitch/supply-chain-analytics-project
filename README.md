# Supply Chain Analytics Dashboard (SQL + Power BI)

## Dashboard Preview (till phase 4)
Built an interactive Power BI dashboard visualizing supplier performance, on-time delivery rates, logistics delays, and 
shipment cost trends across transport modes.
![Dashboard](Dashboard_Preview.png)

Conducted statistical analysis to identify patterns in delivery delays and production downtime. Analyzed root 
causes of 63 extreme downtime events across the supply chain.
